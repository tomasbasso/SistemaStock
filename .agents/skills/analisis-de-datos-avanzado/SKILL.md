# 📊 Análisis de Datos Avanzado

## Descripción General
Esta skill permite transformar datos crudos en información accionable para el negocio (ej. tendencias de stock, reportes de ventas, análisis de costos).

## Flujo de Análisis
1. **Carga y Exploración**: Identificar tipos de datos, valores nulos y estadísticas básicas (`df.info()`, `df.describe()`).
2. **Limpieza (Data Cleaning)**:
   - Tratar valores nulos (imputación o eliminación).
   - Eliminar duplicados.
   - Conversión de tipos (Fechas, Categorías).
   - Manejo de outliers (valores atípicos).
3. **Análisis Estadístico**: Agrupaciones (`groupby`), correlaciones y tablas dinámicas.
4. **Visualización**: Crear gráficos (Histogramas, Boxplots, Series Temporales) para identificar patrones visualmente.

## Herramientas Preferidas
- **Python**: Pandas, NumPy, Matplotlib, Seaborn.
- **SQL**: Consultas de agregación avanzada para análisis directo en BD.

## Cuándo usar esta Skill
- Para generar reportes mensuales de productos más vendidos.
- Para detectar anomalías en los movimientos de inventario.
- Para proyectar necesidades de compra basadas en el historial de stock.

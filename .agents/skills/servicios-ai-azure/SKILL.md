# 🤖 Servicios de IA en Azure

## Descripción General
Esta skill facilita la integración de capacidades de IA en aplicaciones. Se centra en Azure AI Search, Azure Speech y el ecosistema de Azure Cognitive Services.

## Servicios Principales
1. **Azure AI Search (`azure-search`)**: Para implementar búsqueda semántica avanzada y filtrado complejo de datos.
2. **Azure Speech (`azure-speech`)**: Conversión de voz a texto y texto a voz con voces neuronales.
3. **Cognitive Services**: Visión, Lenguaje y Decisión.

## Uso del MCP de Azure
Si el servidor MCP de Azure está activo, priorizar el uso de las herramientas integradas para listar y gestionar estos recursos:
- `mcp_azure_mcp_resources_list` (filtrando por tipo de IA).
- Consultar cuotas y regiones disponibles para modelos específicos.

## Cuándo usar esta Skill
- Cuando el usuario quiera añadir "búsqueda inteligente" a su catálogo de productos.
- Para implementar comandos por voz en la aplicación móvil.
- Para analizar sentimientos o extraer entidades de documentos cargados.
